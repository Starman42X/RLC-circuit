# RLC-circuit
This is an interactive Python file to use for the RLC circuit lab report at HSO

## What you need:

### Two csv files containing your measurements formated like the following:
f / Hz	I / mA	phi / °	UR / V	ULC / V

### Your component specifications:
Your Voltage [V], your resistance [Ω], your inductance [H], your capacities [μF]
